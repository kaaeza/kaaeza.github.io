$ cat info.txt 
ZASM is an emulator of the Zilog Z80 CPU, written in C and designed to run entirely from the terminal.

This program doesn't just execute Z80 Assembly instructions â€” it also displays real-time updates of the CPU registers, memory, and flags, offering a clear and educational look into how the processor works internally.

The goal of this project is to replicate the behavior of the original Z80 as closely as possible, while keeping the code readable and maintainable. It's a great tool for learning about low-level architecture, emulation, and assembly programming.

$ cat status.txt 
As of June 8th, 2025, the Z80 emulator is in Pre-Alpha development.
The full set of 8-bit arithmetic and logic instructions is implemented.
All 8-bit LD instructions between registers and memory are supported.
The emulator features a fully modifiable 64KB memory space.
Further instruction sets and optimizations are in progress.
